# Matt’s Gallery

A digital gallery of handcrafted wooden objects inspired by tradition, material honesty, and timeless design.

https://matt-gallery.vercel.app/

---

## About

Matt’s Gallery is a collection-based website showcasing handcrafted wooden objects.

Each collection explores a different theme:
- Ritual and balance
- Cultural storytelling
- Nordic craftsmanship
- Jewellery and wearable objects
- Upcycled materials and transformation

The focus is on material, form, and the quiet presence of objects in everyday life.

---

## Collections

- Artifacts — timeless games and crafted objects
- Asia — ritual, balance, and everyday grace
- Congo — story-driven carved objects and tradition
- Jewellery — wearable wooden forms
- Upcycled — reassembled materials and transformation
- Viking — Nordic-inspired tools and vessels

---

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Google Fonts (Cormorant Garamond, Inter)
- Deployed on Vercel

---


## Design Principles

- Material-first design
- Minimal UI, focus on objects
- Editorial layout with large typography
- Soft contrast and generous spacing
- Each collection has its own philosophical tone

---

## Purpose

This project explores how digital presentation can reflect physical craftsmanship — turning objects into visual stories rather than product listings.
